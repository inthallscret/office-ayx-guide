# office-ayx-guide

本仓库用于归档与发布多个独立 HTML 页面，内容涵盖办公场景下的参考指南与实用工具页面。所有页面均为静态 HTML，不依赖后端服务，可直接在浏览器中打开使用。

## 项目简介

`office-ayx-guide` 是一个面向办公需求的页面集合仓库。我们收集、整理并维护一系列独立 HTML 文件，旨在为用户提供清晰、可离线使用的参考页面。这些页面不指向任何特定网站或域名，仅作为本地或托管环境下的文档资源。

## 目录说明

```
office-ayx-guide/
├── pages/           # 存放所有 HTML 页面文件
├── assets/          # 静态资源（图片、样式、脚本等）
├── README.md        # 本文件
└── index.html       # （可选）入口索引页面
```

- `pages/`：核心目录，每个 HTML 文件为一个独立页面，文件名反映其内容主题。
- `assets/`：存放页面所需的共用资源，如样式表、图标、字体等。
- `index.html`：若存在，则为浏览目录的起始页，提供页面列表导航。

## 页面归档说明

- 每个 HTML 页面均为自包含文件，尽量将依赖的资源内联或引用自 `assets/` 目录。
- 页面内容以办公辅助、信息参考、工具说明为主，不包含任何商业推广或营销信息。
- 归档的页面可能不定期更新，以修正错误或优化展示效果。
- 历史版本可参考 Git 提交记录。

## 维护说明

- 欢迎提交 issue 报告页面问题或提出改进建议。
- 如需贡献新页面，请将 HTML 文件放入 `pages/` 目录，并在 `index.html`（若存在）中添加链接。
- 请确保提交的页面内容不侵犯他人权益，不包含敏感或违法信息。
- 本仓库不保证页面内容的长期可用性，维护者将尽力保持仓库整洁与更新。

## 使用方式

1. 克隆或下载本仓库。
2. 直接用浏览器打开 `pages/` 下的 HTML 文件，或通过 `index.html` 导航。
3. 所有页面无需网络连接即可使用（除非页面内嵌了外部资源）。
